# CodeWars Assignment

A collection of Python solutions to CodeWars challenges.

## Solutions Included

- **Even or Odd** - Determine if a number is even or odd
- **Convert a Number to a String** - Convert numeric values to string format
- **Remove Spaces from a String** - Remove all spaces from a given string using the `replace()` method
- **Count Vowels** - Count the number of vowels in a string

## Usage

Each function is self-contained and can be imported or run individually:

```python
from codewars_solutions import even_or_odd, number_to_string, no_space, get_count

# Examples
print(even_or_odd(4))           # Output: "Even"
print(number_to_string(123))    # Output: "123"
print(no_space("hello world"))  # Output: "helloworld"
print(get_count("hello"))       # Output: 2
```
